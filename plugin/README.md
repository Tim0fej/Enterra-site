# EnterraAuth — Paper-плагин для Minecraft 1.21.4

Плагин связывает сервер с сайтом Enterra: при входе проверяется регистрация, а код с сайта вводится командой `/code`.

## Как это работает

1. Игрок регистрируется на сайте с **тем же ником**, что в Minecraft.
2. В личном кабинете получает код доступа.
3. Заходит на сервер — плагин спрашивает сайт, зарегистрирован ли ник.
4. Если не зарегистрирован — кик с ссылкой на регистрацию.
5. Если зарегистрирован, но код ещё не введён — игрок заморожен до команды `/code`.
6. `/code ТВОЙ_КОД` — плагин отправляет код на сайт (`POST /api/minecraft/verify`).

## Сборка

Требуется Java 21 и Maven:

```bash
cd plugin
mvn package
```

JAR появится в `plugin/target/EnterraAuth.jar`.

## Установка на сервер

1. Скопируй `EnterraAuth.jar` в папку `plugins/` Paper 1.21.4.
2. Запусти сервер — создастся `plugins/EnterraAuth/config.yml`.
3. Настрой `config.yml`:

```yaml
api-url: "https://твой-сайт.ru"      # URL API (тот же хост, где крутится server/index.ts)
api-key: "enterra-plugin-key"        # = MINECRAFT_API_KEY на сайте
website-url: "https://твой-сайт.ru"  # ссылка в сообщениях игрокам
```

4. На сайте задай переменную окружения (или в `.env`):

```
MINECRAFT_API_KEY=enterra-plugin-key
```

5. Перезапусти сервер.

## API сайта (уже есть в проекте)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/minecraft/check/:username` | Регистрация и верификация |
| POST | `/api/minecraft/verify` | Проверка кода `{ "username", "code" }` |

Заголовок: `x-api-key: <ваш ключ>`

## Команды

- `/code <код>` — ввести код с сайта (алиас: `/verify`)

## Права

Плагин не использует permissions — все игроки проходят проверку одинаково.
